# Avatar

The Avatar component displays user profile pictures, initials, or icons in a consistent, styled container. It supports multiple variants, sizes, and can display images, text, or custom content.

The simplest avatar displays text content with a colored background.

## Variants

Avatars support two shape variants: `default` (rounded square) and `circular` (fully rounded).

<CodeExample codes={VariantCodes} />

## Sizes

Avatars come in three predefined sizes: `sm` (32px), `md` (40px), and `lg` (48px).

<CodeExample codes={SizeCodes} />

## Images

Display user profile images by providing a `source` prop. The image will automatically fill the avatar container.
